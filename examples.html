作例ギャラリー — 設定レシピ付き

FlowArrowで作れる流れる矢印GIFのサンプル集です。各カードの「レシピ」どおりに設定すれば、同じものがすぐ作れます。GIFはすべて白のフチ合わせ色で書き出した透過GIFで、白背景のスライドにそのまま貼れる状態です(画像を右クリック→保存でそのまま使ってもOKです)。

直線フロー(青)

プリセット: 直線 / 動き: フロー / スタイル: 破線 / 色: #4f8cff / 太さ: 16 / 間隔: 広め
用途: システム構成図のデータフロー、AからBへの単純な流れの強調に。

M字カーブ(グレー)

プリセット: M字 / 動き: フロー / スタイル: 破線 / 色: #5f6368 / 太さ: 14
用途: 行ったり来たりする往復プロセス、紆余曲折を経るフローの表現に。Googleスライドのテーマに馴染むグレーです。

S字カーブ(緑)

プリセット: S字 / 動き: フロー / スタイル: 破線 / 色: #34a853 / 太さ: 14
用途: 成長曲線・改善サイクルなどポジティブな流れの演出に。緑は「進行中・成功」の印象を与えます。

伸びるカーブ(赤・描画モード)

プリセット: フック / 動き: 描画 / スタイル: 実線 / 色: #ea4335 / 太さ: 14
用途: 「ここに注目」の指し示し、結論への誘導に。線が伸びて対象を指すので視線誘導が強力です。
ダーク背景のスライドで使う場合: これらのサンプルは白背景用(フチ合わせ色=白)です。ダークテーマのスライドに貼る場合は、FlowArrowでフチ合わせ色をスライドの背景色に設定して書き出し直すと、フチが目立ちません。詳しくは使い方ガイドをご覧ください。
